定义：输出不同的颜色对应打开不同的继电器
        Y0 Y1 Y2 Y3
红色   1  0   0   0        01
黄色   0  1   0   0        02
蓝色   0  0   1   0        04
绿色   0  0   0   1        08  
白色   0  0   1   1        0B
  
指令：③：设置多个继电器开关
01 0F 00 00 00  01 __  __   红
01 0F 00 00 00  02 __  __   黄
01 0F 00 00 00  04 __  __   蓝
01 0F 00 00 00  08 __  __   绿
01 0F 00 00 00  0B __  __   白
